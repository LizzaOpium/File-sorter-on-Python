# 📂 SHINDOUS File Sorter v1.0.0

Простой консольный скрипт для сортировки файлов по папкам, написанный на Python, используйте с осторожностью.

## 🚀 Возможности
- **Умная фильтрация:** Автоматически отсекает папки, обрабатывая исключительно файлы.
- **Мультикатегорийность:** Поддерживает автоматическую сортировку по 6 основным категориям (Документы, Картинки, Установщики, Тексты, Аудио, Видео).
- **Секция "Other":** Все файлы с неизвестными скрипту расширениями складываются в отдельную безопасную папку, ничего не теряя.
- **Регистронезависимость:** Расширения `.PNG`, `.Png` и `.png` обрабатываются одинаково корректно.
- **Полное логирование:** В процессе работы выводит отчет о каждом перемещении, а в конце показывает общую статистику по найденным объектам.

## 🧱 Модульность и кастомизация

Скрипт спроектирован по принципу модульной конфигурации Data-Driven. Вся логика распределения файлов полностью отделена от исполняемого кода и завязана на один центральный словарь `EXTENSIONS`. 

Это дает коду высокую гибкость:
- **Изоляция логики:** Вам не нужно менять алгоритмы циклов или условия `if/else`, чтобы изменить правила сортировки.
- **Безопасность расширения:** Добавление новых типов данных не может сломать архитектуру программы.
- **Легкая интеграция:** Благодаря такой структуре, данный скрипт можно встроить в любую другую консольную оболочку (например, как независимый модуль для `SHINDOUS OS`) буквально одной функцией.

## 🛠 Настройка и добавление своих расширений

Вы можете легко адаптировать сортировщик под свои профессиональные задачи. 

Чтобы добавить новую категорию или новое расширение, достаточно отредактировать словарь `EXTENSIONS` в начале файла.

### Пример добавления новых категорий:
```python
EXTENSIONS = {
    'Documents': ['.pdf', '.docx', '.txt', '.xlsx', '.doc'],
    'Images': ['.jpg', '.jpeg', '.png', '.gif'],
    # Добавление новой категории для архивов:
    'Archives': ['.zip', '.rar', '.7z', '.tar'],
    # Добавление новой категории для разработчиков:
    'Development': ['.py', '.js', '.html', '.css', '.json']
}
```
*Примечание: При добавлении расширений всегда указывайте точку в начале (`.py`, а не `py`), так как системный модуль `os.path.splitext()` извлекает их именно в таком формате.*

## 📦 Требования и Запуск

Скрипт использует исключительно стандартную экосистему модулей Python (`os`, `shutil`) и не требует установки сторонних библиотек через `pip`.

1. Склонируйте репозиторий или скачайте файл `sorter.py`.
2. Запустите скрипт через терминал:
```bash
python sorter.py
```
3. Введите полный путь к папке, в которой необходимо навести порядок (например: `C:\Users\Admin\Downloads`).
